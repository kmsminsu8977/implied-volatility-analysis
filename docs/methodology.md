# 방법론 노트

## 1. 프로젝트 목적

시장 옵션가격에서 Black-Scholes 내재변동성을 역산하고 smile 구조를 해석한다.

## 2. 핵심 질문

같은 만기에서 행사가별 시장가격은 어떤 변동성 구조를 암시하는가?

## 3. 입력 데이터

- 파일: `data/sample/market_option_quotes.csv`
- 성격: 강의자료나 외부 원천을 복제하지 않은 합성 샘플
- 목적: 계산 흐름과 결과 해석을 작은 표본으로 먼저 검증

## 4. 계산 절차

1. 관측 가격을 입력으로 두고 이분법으로 Black-Scholes 내재변동성을 찾는다.
2. moneyness 구간별로 변동성 차이를 비교해 smile/skew의 직관을 정리한다.
3. 가격 데이터는 공개 강의자료가 아니라 임의 생성한 학습용 quote다.

## 5. 산출물 해석

`outputs/tables/baseline_results.csv`는 프로젝트별 baseline 결과입니다. 이 표는 최종 연구 결론이 아니라, 이후 실제 데이터 연결과 상세 분석을 시작하기 위한 구조 검증용 산출물입니다.

## 6. 한계

- 합성 데이터 기반이므로 통계적 유의성이나 투자 가능성을 주장하지 않습니다.
- 거래비용, 세금, 슬리피지, 유동성 제약은 일부 프로젝트에서만 단순 반영했습니다.
- 프로젝트별 정교한 모델링은 별도 후속 작업에서 확장해야 합니다.
